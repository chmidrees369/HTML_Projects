# The Periodic Table

This project displays a visual representation of the periodic table of elements using HTML. The table is organized into rows and columns, showcasing each element's atomic number and symbol, with distinct color coding for different groups of elements.

## Features

- **Responsive Design:** The periodic table is centered on the page and adjusts to different screen sizes.
- **Color Coding:** Elements are color-coded based on their classifications:
  - **Light Pink:** Alkali and alkaline earth metals.
  - **Yellow:** Nonmetals and metalloids.
  - **Light Blue:** Transition metals.
  - **Light Green:** Lanthanides and actinides.
- **Comprehensive Layout:** Contains all 118 known elements, with their respective atomic numbers and symbols.

## HTML Structure

- **DOCTYPE and Language:** The document is defined as an HTML5 document with English as the primary language.
- **Head Section:**
  - Sets the character encoding to UTF-8.
  - Ensures compatibility with Internet Explo
