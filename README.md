**This repository contains a simple HTML project designed to demonstrate the core features and structure of HTML. It serves as a foundational template for beginners to learn how to create web pages using HTML.

### Features

- **Semantic HTML:** Utilizes HTML5 elements for improved accessibility and SEO.
- **Basic Structure:** Includes a well-organized layout with header, footer, and main content sections.
- **Form Elements:** Demonstrates the use of various form inputs, including text fields, radio buttons, and checkboxes.
